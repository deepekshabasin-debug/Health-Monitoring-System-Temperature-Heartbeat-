# Health-Monitoring-System-Temperature-Heartbeat-
Proteus simulation of an embedded, real-time health monitoring system. Features an LM35 temperature sensor and a heartbeat pulse sensor interfaced with a microcontroller to display vital signs on a 16x2 LCD. Ideal for biomedical students and firmware developers looking to test hardware virtually without physical prototypes.
## Project Overview

This repository hosts a comprehensive **Proteus simulation project** designed to model an embedded, real-time health monitoring system. Tailored for biomedical engineering students, hobbyists, and firmware developers, this virtual prototype showcases how hardware components interface with a microcontroller to track critical human vital signs dynamically and safely without requiring physical hardware prototyping.

### Key Features
* **Temperature Monitoring:** Utilizes an LM35 sensor to accurately track simulated body temperature changes.
* **Heart Rate Tracking:** Simulates a pulse sensor to compute real-time beats per minute (BPM).
* **LCD Output:** Displays all calculated vital data clearly on a virtual 16x2 character LCD screen.
